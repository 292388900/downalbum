准备工作，设置编译器。
打开VC++6.0新建一个Win32 Dynamic-Link Library工程(一个简单的dll工程)，把Plugin.h
头文件加到工程中来，在.cpp文件中include Plugin.h头文件。
打开Project->Settings,在C/C++选项卡的最下面Project Options里加上“/J”（没有引号），
就是将缺省的char类型设置为unsigned（Set default char type to unsigned）。

因为在Plugin.h中需要设置，否则会有错误。
//================================================================================
  #ifndef _CHAR_UNSIGNED               // Verify that character is unsigned
    #error Please set default char type to unsigned (option /J)
  #endif
//================================================================================
打开“工程/增加到工程”把OllyDbg.lib文件加到工程里来。
在编译链接过程中出现LNK2001错误可将Plugin.h头文件中的API函数名称前加下划线"_"，
或者在你的C/CPP文件中将引用的API函数前面加下划线"_"。

更为夸张者，需要将Plugin.h头文件中的API函数和C/CPP文件中引用的API函数前面都要加下划线"_"。
